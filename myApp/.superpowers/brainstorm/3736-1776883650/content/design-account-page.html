Design decisions baked in

  • Mode toggle — only rendered if user has a pros record. Customer-only users never see it.
  • 成為設計師 — shown in Support only when user has no pro account. Hidden for pros.
  • Liked pros row — shows avatar stack as a count preview. Tapping opens the 喜愛的設計師 screen.
  • Tapping 預約 on liked pro → simple sheet (name + photo) → navigates to slots screen with pro pre-selected.
  • Heart button — added to results pro cards + BookingDetailSheet. Toggles liked state optimistically.
  • Edit profile — rendered at 38% opacity, disabled. Placeholder for later.
  • Help / About — both link out (browser or modal). Placeholder routes for now.
